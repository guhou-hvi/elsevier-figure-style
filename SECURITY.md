# Security Policy

## Supported Versions

`elsevier-figure-style` is currently a `v0.1.x` beta. Security fixes are applied to the latest beta release only.

## Reporting a Vulnerability

Do not open a public issue for a vulnerability that could expose local files, execute unintended code, escape a profile bundle, or modify files outside an expected target directory.

After this repository is published, use GitHub's **Security** tab and **Report a vulnerability** to submit a private report. Include:

- the affected script and version;
- a minimal synthetic reproduction;
- the expected and observed behavior;
- the operating system and Python or R version; and
- any suggested mitigation, if known.

If private vulnerability reporting is temporarily unavailable, contact the maintainer through [@guhou-hvi](https://github.com/guhou-hvi) without posting exploit details publicly.

The project is unofficial and does not handle Elsevier credentials or submission-system access. Reports about Elsevier services should be directed to Elsevier rather than this repository.
