# Changelog

## [0.1.0] - 2026-07-15

### Added

- Source-backed Elsevier-style figure profile.
- Matplotlib helpers for line, bar, heatmap, and scatter plots.
- Python source-code and bitmap metadata checkers.
- Visual audit workflow for result and schematic figures.
- Basic R/ggplot2 theme and export support.
- English and Simplified Chinese documentation.
- Synthetic examples and CI validation.

### Current limitations

- This is an unofficial Elsevier-style helper.
- Target-journal author guidelines remain authoritative.
- R support is basic and has no static source checker.
- Schematic figures are audited but not automatically redrawn.
